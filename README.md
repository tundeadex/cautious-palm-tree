# CYPRESS CUCUMBER TYPESCRIPT FRAMEWORK

## Prerequisites

Ensure you have node 18 and above

## Install dependencies

```
npm install

```
## Run all tests

```
npm run all
npx cypress run
```

## Run test with tag

```
npm run uitest
npx cypress run --headed -e TAGS='@test'

npm run apitest
npx cypress run --headed -e TAGS='@apitest'

```

## Create and open report

```shell
npm run report
```
